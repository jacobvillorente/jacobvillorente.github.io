branding

while i don't have an official brand, if you wish to reference the jacob villorente website, use my logo, or otherwise mention elsewhere, please abide by these branding guidelines.

you can find all of these images in this google drive folder

logo

this is the full main logo that you see in the header and the home page. there is one with white text and one with black text. please use each logo so that the text shows up clearly. for example, using the white logo on a dark background, or the black logo on a light background. 

emblem

i'd prefer you to use the full logo, but if your site does not have enough room or space, you can also use the emblem, which is a capital letter j in a round cherry blossom box. 

text

all text used on this site is open sans. the "recently played" was in source sans pro but that has since changed. also, almost all text is in lower case. this is for aesthetic reasons.
